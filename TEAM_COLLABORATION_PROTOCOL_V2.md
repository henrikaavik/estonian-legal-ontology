# Team Collaboration Protocol v2 (Equal Partner Mode)

Status: ACTIVE  
Effective: immediately

## 1) Ühine arutelu enne teostust
- Iga suurema teema puhul toimub mini-arutelu (vähemalt 2 agenti).
- Arutelus kaetakse: eesmärk, variandid, riskid, soovitatud lahendus.

## 2) Võrdne partnerlus
- Keegi ei ole lihtsalt "täitja".
- Iga agent võib vaidlustada lahenduse põhjendatult.
- Kui konsensust pole 10 minutiga: Owner otsustab ja logib põhjenduse.

## 3) Kohustuslik 2-etapiline kvaliteedikontroll
- Stage A: Peer critique (teine agent leiab puudused).
- Stage B: Peer improve (teine agent teeb reaalsed parandused, mitte ainult kommentaarid).
- Alles siis läheb väljund "final" staatusele.

## 4) Handoff standard (kohustuslik)
- Objective
- Context
- Current state
- Risks
- Exact next action
- Done criteria

## 5) Ühine projekti fail
- Igal projektil peab olema üks ühine fail `~/.openclaw/shared/` all.
- See on source of truth: otsused, staatus, järgmised sammud.

## 6) Retrospektiiv
- E/N retrospektiiv kohustuslik.
- Vaadatakse üle viimane periood + 30 päeva trend.
- Parandused rakendatakse kohe ja kirjutatakse faili.
